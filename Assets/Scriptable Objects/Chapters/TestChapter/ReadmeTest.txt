To use as a test chapter:

Make sure test scene has the LevelGeo object has the spawner attached as a child.
On the spawner child needs to be the block spawner script.
Fill the Chapter variable on the spawner script with the TestChapter scriptable object under the TestChapter folder within the scripable objects folder.
On the TestChapter scriptable object fill in the Sentences array variable with any sentences needing to be tested.
Always make sure the last variable in the sentences array is the LastSentence object under the Chapters folder.
Play the test scene and enjoy ease of testing.
